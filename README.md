# Noroff VueApp

## Project setup
The repository does not contain the node_modules, but it does contain the package.json file. Install the modules to get all the dependencies.
```
npm install
```

### Compiles and hot-reloads for development
Start a development server for the project with the serve script, this loads a webserver on localhost:8080 by defualt.
```
npm run serve
```

### Compiles and minifies for production
Run the build script to compile a production ready set of files.

```
npm run build
```
